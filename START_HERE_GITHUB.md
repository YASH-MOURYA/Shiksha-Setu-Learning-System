## ⚡ Super Quick Start (2 Minutes)

### Option 1: Automated (Easiest) ✨

**Just double-click this file:**
```
PUSH_TO_GITHUB.bat
```

That's it! The script will guide you through everything.

---

### Option 2: Manual (5 Minutes) 🛠️

**Copy and paste these commands:**

```bash
# 1. Open terminal in project folder
cd C:\Users\yash7\Downloads\New-S-S\shiksha-setu

# 2. Initialize Git
git init

# 3. Add all files
git add .

# 4. Create first commit
git commit -m "Initial commit: Complete LMS project"

# 5. Connect to GitHub (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/shiksha-setu.git

# 6. Push to GitHub
git branch -M main
git push -u origin main
```

**Done! Your code is now on GitHub! 🎉**

## 📞 Need Help?

### Quick Solutions:

**Problem: Git not installed**
- Download from: https://git-scm.com/download/win

**Problem: Don't have GitHub account**
- Sign up at: https://github.com/signup

**Problem: Repository not found**
- Make sure you created the repository on GitHub first
- Check the repository name matches exactly

**Problem: Authentication failed**
- Use Personal Access Token instead of password
- Guide: https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/creating-a-personal-access-token
